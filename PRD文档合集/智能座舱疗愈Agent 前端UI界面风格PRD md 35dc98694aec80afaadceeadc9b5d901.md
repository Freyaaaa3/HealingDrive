# 智能座舱疗愈Agent 前端UI界面风格PRD

**文档版本**：v1.0 Demo
**文档状态**：正式PRD
**更新日期**：2026-05-11
**设计融合**：苹果原生UI设计语言 + 蔚来汽车助手面板视觉规范

## 一、模块概述

本文档定义**智能座舱疗愈Agent** 全部前端界面视觉风格、布局规范、色彩体系、字体层级、控件样式、动效逻辑、页面适配规则。
整体设计深度融合两大设计体系：

1. **苹果UI风格**：极简留白、毛玻璃磨砂、低饱和渐变、圆润圆角、轻量化控件、无冗余线条、高质感通透感
2. **蔚来助手面板风格**：车载座舱专属比例、悬浮层级、座舱氛围适配、居中视觉焦点、极简信息排布、车载安全交互逻辑

适用于中控屏2D虚拟形象面板、疗愈交互弹窗、语音悬浮层、全局控件、空态/异常态全页面统一视觉规范。

## 二、整体设计核心原则

| 设计原则 | 详细规范 |
| --- | --- |
| 极简留白 | 沿用苹果UI大留白逻辑，拒绝冗余分割线、冗余图标，元素呼吸感充足 |
| 磨砂通透 | 全局采用苹果毛玻璃磨砂质感背景，适配座舱壁纸不突兀 |
| 圆润几何 | 统一大圆角曲率，卡片/按钮/弹窗全部同弧度，苹果极简几何美学 |
| 车载居中聚焦 | 遵循蔚来助手面板设计，核心虚拟形象居中置顶，视觉焦点集中 |
| 低饱和高级色 | 配色参考苹果浅色模式+蔚来车载低饱和莫兰迪色系，不刺眼、适配行车视觉 |
| 轻量化动效 | 苹果微交互逻辑，渐变呼吸、轻微浮动，无夸张动画，不干扰驾驶 |
| 无干扰设计 | 所有界面不遮挡导航、车速、限速等行车核心信息，遵循蔚来车载安全规范 |
| 统一层级 | 悬浮层、卡片层、背景层严格分层，对齐弹窗层级逻辑 |

## 三、布局与结构规范

### 3.1 页面通用布局结构

完全沿用**蔚来助手面板经典三段式布局** + 苹果无边界极简设计：

1. **顶部区**：极简留白，无标题栏、无多余图标
2. **核心视觉区**：居中放置2D疗愈虚拟形象（蔚来式视觉焦点设计）
3. **底部区**：极简语音状态提示，无功能按钮、无触控控件（纯语音交互）

布局特征：

- 整体居中对称布局，适配中控横屏/竖屏
- 左右大留白，复刻苹果UI松弛感
- 无分割线、无边框，依靠磨砂层级区分内容
- 整体面板悬浮于座舱桌面之上，半透明沉浸感

### 3.2 圆角规范

全局统一圆角曲率，**苹果大圆角风格** + 蔚来车载面板同款弧度：

- 悬浮面板整体圆角：**28dp**
- 内部元素/小卡片圆角：**16dp**
- 按钮/标签圆角：**12dp**
- 禁止直角、禁止过小弧度，全页面曲率统一

### 3.3 层级划分（对齐苹果弹窗层级）

1. **底层**：座舱原生壁纸（完全透出，不遮死）
2. **磨砂悬浮层**：疗愈Agent主面板（半透明毛玻璃）
3. **核心元素层**：2D虚拟形象、状态文案
4. **高亮层**：动效呼吸光、状态微光点缀

层级逻辑：不做厚重蒙版遮挡，采用苹果通透式分层，蔚来助手同款沉浸感。

## 四、色彩系统

### 4.1 主色调体系

融合苹果浅色极简风 + 蔚来治愈系车载配色：

- **主色**：柔和雾霾蓝（低饱和，不刺眼）
- **辅助色**：浅奶白、莫兰迪灰、淡青微光
- **文字主色**：深灰黑（非纯黑，苹果阅读舒适感）
- **文字次要色**：浅灰，弱化不抢视觉
- **动效光色**：淡蓝微光呼吸效果

### 4.2 背景配色规范

1. 主面板背景：**苹果毛玻璃磨砂白**
半透明磨砂 + 轻微模糊，适配任意座舱壁纸，不抢风头
2. 禁止使用：高饱和纯色、厚重黑色蒙版、渐变艳色
3. 全局遵循：浅色极简基调，适配日间/夜间座舱自动适配亮度

### 4.3 状态色规范

- 待机：浅灰微光
- 倾听：淡蓝微弱呼吸光
- 语音播报：轻微青色调柔光
- 异常/空态：保持低调灰色，无警示大红大紫

## 五、字体规范

采用**苹方/旧金山苹果系无衬线字体** + 蔚来车载极简字重规范：

1. 标题/核心状态字：中等字重，字号18dp
2. 辅助提示字：常规字重，字号14dp
3. 字重规范：仅区分常规/中等，无粗黑体，保持苹果轻盈感
4. 文案风格：极简短句，无长文案、无冗余描述
5. 文字对齐：全部居中对齐，贴合蔚来居中设计语言

## 六、控件与元素风格

### 6.1 全局控件规则

因产品**纯语音无触控**，界面无按钮、无输入框、无图标控件，
仅保留**极简状态标签**，风格遵循：

- 苹果胶囊式标签、无边框、浅磨砂底色
- 无图标堆砌，只用极简文字状态

### 6.2 2D虚拟形象展示规范

1. 形象尺寸：居中占面板核心区域，上下留白充足（蔚来焦点构图）
2. 背景：无额外相框、无装饰边框，苹果无边界设计
3. 动效风格：采用苹果轻微呼吸浮动，低速平缓上下微动
4. 禁止：夸张肢体动作、高闪特效、二次元浓配色

### 6.3 磨砂毛玻璃规范

全局复刻苹果iOS磨砂质感：

- 模糊强度：中度模糊，保留壁纸轮廓又不杂乱
- 透明度：75%通透感
- 无阴影硬边框，依靠层级自然区分
- 适配座舱昼夜模式自动调节明暗

## 七、动效设计规范

全部采用**苹果微交互理念** + 蔚来沉稳车载动效：

1. **面板弹出动效**：轻微从上往下平缓划入，渐入透明，无弹跳
2. **待机动效**：缓慢呼吸浮动，节奏舒缓，不打扰驾驶
3. **倾听动效**：头像轻微点头，低帧率柔和动画
4. **语音播报**：底部文字渐显渐隐，无闪烁
5. **面板收起**：平缓渐隐缩小，顺滑退场

硬性约束：

- 所有动效低帧率、低幅度，无快闪、无震动、无夸张形变
- 全程安静柔和，符合车载疗愈氛围

## 八、页面适配规范

1. 适配主流车载中控横屏、竖屏，自动居中留白
2. 异形屏自动避让刘海/挖孔，不遮挡面板
3. 分辨率自适应，等比缩放不拉伸变形
4. 始终保持居中构图，严格遵循蔚来助手面板视觉重心
5. 夜间模式自动降低亮度饱和度，延续苹果夜间护眼逻辑

## 九、全局页面视觉一览

### 9.1 唤醒主面板

- 苹果磨砂悬浮圆角面板
- 居中2D虚拟形象（蔚来焦点式布局）
- 底部极简居中状态文案：聆听中/正在疗愈
- 无任何按钮、无图标、无多余装饰
- 大留白、通透沉浸感

### 9.2 空态/异常态

- 保留同款磨砂面板样式
- 居中简约文字提示
- 无报错红色警示、无弹窗警告
- 维持整体高级极简风格不变

### 9.3 会话中状态

- 形象切换倾听/说话动效
- 底部文案柔和渐变切换
- 无界面变化，不额外弹出新窗口

## 十、禁止设计规范

1. 禁止厚重深色蒙版、高饱和大色块
2. 禁止直角卡片、粗分割线、冗余UI元素
3. 禁止夸张动效、闪光特效、弹窗广告式设计
4. 禁止复杂多按钮布局、违背纯语音交互逻辑
5. 禁止遮挡导航、车速、行车核心区域
6. 禁止二次元浓配色、卡通浮夸风格
7. 禁止多文字堆砌，保持苹果极简短句风格

## 十一、设计融合总结

整体UI最终呈现效果：
**蔚来汽车助手面板的车载居中沉稳布局 + 苹果UI的极简留白、磨砂通透、圆润高级感、轻量化动效**
无冗余元素、无触控按钮、视觉克制柔和，完全适配智能座舱驾驶场景，同时具备高端产品质感。
